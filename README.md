# Customer Shopping Behavior Analysis

## Project Overview
This project analyzes customer shopping behavior to understand purchasing patterns, product performance, customer loyalty, and the impact of discounts.  
The analysis was performed using **Python, SQL, and Power BI**.

## Tools Used
- Python (Pandas, Jupyter Notebook)
- MySQL
- Power BI
- GitHub

## Dataset
The dataset contains information about:
- Customer demographics
- Products purchased
- Purchase amount
- Discounts
- Customer loyalty
- Shipping preferences
- Review ratings

## Data Processing
Data cleaning and feature engineering were performed using **Python in Jupyter Notebook**.

Main steps:
- Handling missing values
- Creating age groups
- Converting purchase frequency to numeric values
- Preparing data for SQL and Power BI analysis

## SQL Analysis
SQL was used to answer key business questions such as:

- Revenue contribution by gender
- Top products by category
- Customer loyalty segmentation
- Impact of discounts on purchases
- Subscription behavior of repeat buyers

## Dashboard (Power BI)
A Power BI dashboard was created to visualize key insights including:

- Total revenue
- Customer distribution
- Revenue by category
- Revenue by age group
- Discount impact on spending
- Top selling products
- Customer loyalty segments

## Key Insights
- Certain product categories generate significantly higher revenue.
- Loyal customers contribute a large portion of total sales.
- Discounts influence purchasing behavior.
- Demographics play a role in spending patterns.
